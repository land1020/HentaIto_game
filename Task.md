# タスク: ロビー「お題モード」設定の拡張

「お題モード」の「自動選出」選択時に、「オリジナル(親入力)」オプションを追加し、自動選出されたお題が気に入らない場合に親が手動で入力できるようにします。

## 実装
- [x] 型定義（Type）の更新
    - `GameSettings` に `allowOriginalInAuto` フラグを追加
- [x] ロビー画面（UI）の更新

- [x] テーマ選択画面（ThemeSelectionScreen）の更新
    - AUTOモードかつ `allowOriginalInAuto` がONの場合、オリジナル入力フォームを表示
